## 1. 设计规约 (Design Conventions)

### 1.1 命名规范 (Naming Convention)

* **表名** ：使用 **复数形式** ，小写下划线分隔（`snake_case`）。例：`projects`, `video_prompts`。
* **主键** ：统一命名为 `id`，类型为 `UUID`。
* **外键** ：`[单数表名]_id`。例：`project_id`, `scene_id`。
* **索引** ：`idx_[表名]_[字段名]` 或 `uq_[表名]_[字段名]` (唯一索引)。
* **时间字段** ：统一使用 `created_at` (创建时间) 和 `updated_at` (更新时间)。

### 1.2 核心策略 (Core Strategy)

* **ID 双轨制** ：
  * **物理主键 (PK)** ：使用 `UUID v4`，用于系统内部关联，确保全局唯一且无序。
  * **业务编码 (Code)** ：对外展示/人类阅读字段（如 `role.name`、`permission.code`），并以唯一约束保证一致性。
* **JSONB 使用原则** ：
  * 易变结构存储为 JSONB（如 `audit_logs.meta`）。
  * 核心查询字段必须提升为实体列（如 `audit_logs.action`、`permissions.code`）。
* **软删除 (Soft Delete)** ：权限与审计模块当前不启用软删除；如未来涉及合规与恢复需求，可为核心表增加 `deleted_at`。

### 3.16 Agent 表 (agents)

说明：可配置的业务 Agent（文本/图片/视频）。Agent 直接绑定到一个 `ai_model_configs` 记录，运行时按该配置进行调用并按 `credits_per_call` 扣积分。

| **字段名** | **类型** | **必填** | **默认值** | **描述/约束** |
| --- | --- | --- | --- | --- |
| `id` | `UUID` | ✅ | `gen_random_uuid()` | 物理主键 |
| `name` | `VARCHAR(128)` | ✅ | - | Agent 名称，唯一 |
| `category` | `VARCHAR(16)` | ✅ | - | 类别（`text/image/video`） |
| `purpose` | `VARCHAR(32)` | ✅ | `'general'` | 用途（如 `storyboard_extraction/asset_extraction/.../general`） |
| `ai_model_config_id` | `UUID` | ✅ | - | 模型配置（FK -> `ai_model_configs.id`，on delete restrict） |
| `capabilities` | `JSONB` | ✅ | `[]` | 能力集合（如 `["text"]` / `["image"]` / `["video"]`） |
| `system_prompt` | `TEXT` | - | - | 系统提示词 |
| `user_prompt_template` | `TEXT` | - | - | 用户提示词模板（如 `{input}`） |
| `credits_per_call` | `INT` | ✅ | `0` | 单次调用消耗积分 |
| `enabled` | `BOOLEAN` | ✅ | `true` | 是否启用 |
| `created_at` | `TIMESTAMP` | ✅ | `now()` | 创建时间 |
| `updated_at` | `TIMESTAMP` | ✅ | `now()` | 更新时间 |

约束：
* `UNIQUE uq_agents_name (name)`
* `CHECK ck_agents_category`：`category` 必须为 `text/image/video`
* `CHECK ck_agents_purpose`：`purpose` 必须为预置用途集合
* `CHECK ck_agents_credits_per_call_nonneg`：`credits_per_call >= 0`

索引：
* `INDEX idx_agents_category (category)`
* `INDEX idx_agents_purpose (purpose)`
* `INDEX idx_agents_enabled (enabled)`
* `INDEX idx_agents_model_config (ai_model_config_id)`
