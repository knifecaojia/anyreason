## 📋 设计概述

将文本、图片和视频模型测试统一为一个 UI，以图片测试为蓝本，整合所有功能到统一的对话框中。

### ✨ 核心改进

1. **统一入口** - 文本/图片/视频共享同一套 UI
2. **单一显示** - 缩略图只显示在对话框内部上侧
3. **动态适配** - Placeholder 和逻辑根据类型自动切换
4. **简洁交互** - 移除重复显示，保持界面清爽

## 🔧 技术实现

### 1. 统一条件判断

```typescript
// 支持三种类型
{activeModelTab === "image" || activeModelTab === "text" || activeModelTab === "video" ? (
  <UnifiedTestUI />
) : null}
```

### 2. 动态会话创建

```typescript
// 根据当前类型创建会话
onClick={() => void createModelTestSession({ 
  category: activeModelTab,  // 动态类型
  aiModelConfigId: modelTestModelConfigId 
})}
```

### 3. 动态 Placeholder

```typescript
placeholder={
  activeModelTab === "text" ? "请输入提示词..." :
  activeModelTab === "video" ? "请输入视频生成提示词，输入 @ 引用参考图..." :
  "请描述你想生成的图片，输入 @ 引用参考图"
}
```

### 4. 缩略图单一显示

**位置：** 对话框内部上侧（padding 内部）

```tsx
{/* 缩略图回显区域（对话框内部上侧） */}
{modelTestSessionImageAttachmentNodeIds.length > 0 && (
  <div className="px-4 pt-4 pb-2">
    <div className="flex items-center gap-2 overflow-x-auto pb-2">
      {modelTestSessionImageAttachmentNodeIds.map((id, idx) => (
        <Thumbnail key={id} id={id} index={idx + 1} />
      ))}
      {/* 快速上传按钮 */}
      <label className="upload-button">
        <input type="file" accept="image/*" multiple />
        <Plus size={16} />
      </label>
    </div>
  </div>
)}
```

## 🎯 交互流程

### **文本测试**
```
1. 选择"文本"标签
2. 输入提示词
3. 点击发送
4. 查看响应
```

### **图片测试**
```
1. 选择"图像"标签
2. 点击 [+] 上传参考图
3. 缩略图显示在对话框内部上侧
4. 输入提示词，输入 @ 引用参考图
5. 点击缩略图插入 @n
6. 点击发送，生成图片
```

### **视频测试**
```
1. 选择"视频"标签
2. 上传参考图（可选）
3. 输入视频生成提示词
4. 点击发送，生成视频
```

## 💡 设计亮点

### **1. 统一而不失灵活**
- ✅ 共享同一套 UI 框架
- ✅ 根据类型动态调整提示文本
- ✅ 保持各类型的特殊功能（如图片的 @ Mention）

### **2. 简洁而不失功能**
- ✅ 缩略图只显示一处（对话框内部上侧）
- ✅ 快速上传按钮整合到缩略图区域
- ✅ 移除重复的底部缩略图显示

### **3. 直观而不失细节**
- ✅ 已引用图片显示蓝色对勾
- ✅ Hover 时显示删除按钮
- ✅ 序号标签清晰可见

### **4. 高效而不失优雅**
- ✅ 点击缩略图直接插入 @n
- ✅ 输入 @ 弹出悬浮框选择
- ✅ 支持键盘快捷键（Enter 发送，ESC 关闭）

## 🎉 总结

统一的模型测试 UI 设计以图片测试为蓝本，整合了文本、图片和视频三种类型，实现了：

1. **视觉统一** - 共享同一套 UI 框架
2. **功能完整** - 保留各类型的特殊功能
3. **交互简洁** - 移除重复显示，保持清爽

**关键改进：** 缩略图只显示在对话框内部上侧，避免了之前的两处显示问题，界面更加简洁直观！✨
