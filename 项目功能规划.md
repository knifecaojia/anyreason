# AI漫剧创作管理平台｜项目功能规划

本文档依据 `fixed-goal.md` 要求编写，结合 `refs/可梦手册.md` 中可迁移功能，形成本项目的功能规划与流程设计说明。

## 1. 项目定位与目标
- **定位**：面向漫剧创作企业的 AI 协同生产平台，覆盖“创意 → 剧本 → 资产 → 分镜 → 成片”的全流程。
- **目标**：在“轻量化、前后端分离、RBAC 权限控制”的基础上，提供可扩展的 AI 模型编排能力、资产管理与资产可视化能力。

## 2. 用户与角色
- 企业管理者：创建组织/项目、权限分配、成本与进度把控。
- 策划：创意生成、剧本拆解、提示词管理。
- 美术：角色/场景/道具/特效资产设计与一致性把控。
- 导演：分镜节奏、镜头调度、故事板审核。
- 编辑/剪辑：素材导出、成片管理、投放版本管理。

## 3. 核心功能规划（一级模块）
1. **账号与权限（RBAC）**
   - 组织/团队/项目级权限。
   - 角色、资源、动作三级权限控制。
2. **项目生命周期管理**
   - 项目创建、阶段推进、里程碑与状态流转。
3. **剧集基础信息与全局设置（参考可梦）**
   - 屏幕比例（9:16 / 16:9）、风格分类（写实/动漫）。
   - 默认模型选择（LLM/图像/视频/音频）。
   - 全局提示词与模板管理。
4. **AI 模型与 Provider 管理（参考 dify）**
   - 统一模型注册与配置中心。
   - 支持 LLM、图像、视频、音频、多模态。
5. **拖拽式创作画布（无限画布）**
   - 剧本分析、分镜流程、素材链路可视化。
6. **资产管理（角色/场景/生物/道具/特效/音色）**
   - 资产库、版本、关联引用、全局一致性。
   - 资产锁定与分享（保持统一性，允许外部审核）。
   - 资产可视化产物（多视图、多姿态、多风格参考图）。
7. **分镜与故事板**
   - 场景拆分、镜头管理、镜头参数化生成。
8. **团队协作与任务管理**
   - 任务分工、进度跟踪、审核流。
9. **素材导出与成片管理**
   - MP4 / XML / PR 工程等导出。
10. **模型编排与外部工作流衔接**
   - 支持与 ComfyUI / liblib 等工作流编排平台对接。
11. **成本与用量统计**
   - 模型调用量、积分/费用、人员消耗分布。

## 4. AI漫剧从“创意”到“成片”的流程设计
> 本章为平台核心流程章节：从创意生成到成品输出，每个节点阐明平台功能与协作角色。

### 4.0 剧集创建与全局设置（参考可梦）
- **输入**：剧集名称、屏幕比例（9:16 / 16:9）、风格分类（写实/动漫）。
- **平台能力**：
  - 默认模型配置（LLM/图像/视频/音频）。
  - 全局提示词配置与复用（可选模板）。
- **输出**：剧集基础配置、全局提示词与默认模型策略。

### 4.1 创意与题材定位
- **输入**：题材方向、竞品分析、平台偏好。
- **平台能力**：
  - 创意生成器（基于 LLM：剧情梗概、卖点、人设雏形）。
  - 题材标签库与趋势数据库对照。
- **输出**：创意简报、主角设定、情绪走向。

### 4.2 剧本生成与拆解
- **平台能力**：
  - AI 剧本生成（大纲 → 分集 → 分场）。
  - 结构分析（人物、地点、道具、情节节点）。
  - 分场与镜头可视化结构树。
- **输出**：标准化剧本与“镜头级”拆解数据。

### 4.3 角色/场景/生物/道具/特效/音色资产设计
- **平台能力**：
  - 自动识别剧本实体（角色、场景、生物、道具、特效、音色）。
  - 资产提示词生成与统一模板管理。
  - 资产版本锁定（保证一致性）。
  - 资产锁定后进入分镜环节（对齐可梦工作流）。
  - 资产分享链接（用于外部审核/协作）。
- **输出**：资产库 + 统一设计规范 + 资产可视化资源包。

#### 4.3.1 资产可视化与一致性校验
- 为角色、场景、道具、特效生成可视化图片资源（多视角、多姿态、多风格参考图）。
- 可视化资源作为“图生视频/分镜生成”的基础依据，并在镜头生成时强引用。
- 资产可视化支持版本化管理与一致性校验（尺寸、风格、色系、细节稳定性）。

### 4.4 分镜与故事板创作
- **平台能力**：
  - 场景拆分 → 镜头拆解 → 镜头提示词生成。
  - 故事板四区：创意生成区 / 时间线 / 预览区 / 素材区。
  - 局部重绘、扩图、修图、超清、派生镜头。
- **输出**：可执行故事板与镜头资产（绑定资产可视化资源）。

### 4.5 图生视频与配音
- **平台能力**：
  - 图生视频、首尾帧控制、运镜参数。
  - 配音音色库、口型同步、语速语调控制。
- **输出**：镜头级视频与配音素材。

### 4.6 高效版一键流程（参考可梦）
- **平台能力**：
  - 一键完成剧集分析、镜头分析、图片生成、视频生成、旁白生成。
  - 异步任务与进度回写（关闭页面不影响生成）。
- **输出**：高效版“快速成片”物料包。

### 4.7 成片导出与审核
- **平台能力**：
  - 多格式导出（MP4 / XML / PR）。
  - 素材打包导出（分镜素材/故事板素材）。
  - 逐帧批注与审核流。
- **输出**：可发布成片与工程文件。


## 5. AI 设计章节（模型对比与提示词工程管理）

### 5.1 多模型能力对比
- **LLM 对比维度**：
  - 创意质量、结构一致性、分镜可拆解性、成本与速度。
- **图像模型对比维度**：
  - 风格一致性、角色一致性、细节稳定性、可控度。
- **视频模型对比维度**：
  - 运镜自然度、时长稳定性、画面连贯度。
- **音频模型对比维度**：
  - 口型同步、情绪表达、音色可定制性。

平台提供“模型对比面板”，支持同一提示词在多模型下并行产出，并记录指标评分与人工反馈。

### 5.2 提示词工程管理
- **提示词库**：全局提示词 / 资产提示词 / 镜头提示词分层管理。
- **模板化**：固定槽位（如年龄、性别、国籍、服饰、动作）。
- **版本控制**：提示词版本与生成结果绑定。
- **协同审批**：提示词修改需走审批流程，保证一致性。
- **复用策略**：按项目、角色、场景维度复用模板。

### 5.3 外部工作流衔接（ComfyUI / liblib）
- **工作流对接**：支持导入/导出节点式工作流配置，实现平台内外的双向同步。
- **资产映射**：角色/场景/道具可视化资源自动映射为工作流输入参数。
- **提示词映射**：平台提示词模板可绑定工作流节点，保持一致性与可追溯性。
- **执行回流**：支持任务提交、进度回调、结果回流与版本记录。

## 6. AI 在关键环节中的作用

### 6.1 剧本创作
- 生成剧情大纲、分集结构、角色弧线。
- 提供节奏建议与卖点强化。

### 6.2 资产提取与可视化
- 自动识别剧本中角色/场景/道具/特效。
- 一键生成资产描述与提示词。
- 批量生成资产可视化图片（多视角/多姿态/多版本），为后续图生视频提供依据。

### 6.3 分镜创作
- 自动拆分镜头与画面描述。
- 生成镜头提示词与运镜描述。

### 6.4 抽图质检
- 识别不一致角色、错误道具、错位场景。
- 自动打分与缺陷标注（模糊、变形、比例异常）。

### 6.5 配音与口型
- 自动匹配音色与角色。
- 口型同步纠偏与语速调节。

## 7. 资产与协作设计
- **资产锁定机制**：角色/场景/道具锁定后不可随意修改。
- **任务分工**：镜头级任务分配，进度可视化。
- **共享与复用**：跨项目资产库，支持分享和授权。
- **可视化规范**：资产可视化图片统一命名、尺寸与风格基线。
 - **分享流程**：支持对外分享资产列表（含有效期控制）。

## 8. AI 用量与成本统计设计

### 8.1 用量统计
- **维度**：用户、角色、项目、模型、功能模块。
- **指标**：调用次数、token 用量、生成耗时、失败率。

### 8.2 成本核算
- **成本结构**：模型调用费用 + 存储费用 + 渲染费用。
- **成本分摊**：按人员/项目/部门统计。
- **成本预警**：阈值预警与预算限制。

### 8.3 成本报表
- 项目总成本、单集成本、单镜头成本。
- AI 产出效率评估（成本 / 产出比）。

## 9. 技术与实现约束
- **后端**：Python + FastAPI
- **数据库**：PostgreSQL
- **对象存储**：MinIO
- **前端**：Shadcn/ui + Tailwind
- **架构要求**：前后端分离、轻量化、低耦合。
